# Virtual Memory (가상 메모리)
여러개의 process들이 동시에 실행되는 경우 OS는 memory management를 어떠한 방식으로 수행할까?
이를 이해하기 위해서는 virtual memory를 알아야 한다.

## process란
program이 memory에 적재된 상태를 의미한다. 
OS는 process에게 독립적인 메모리(가상 메모리)를 제공한다.
메모리는
- code segment: code(기계어)가 저장된다.
- data segment: 전역변수, 정적변수, 등이 저장된다.
- heap segment: 동적할당시 사용. (new(), malloc())
- stack segment: 지역변수, 함수의 매개변수
